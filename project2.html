Smart Energy Monitor

Sistem monitoring lonjakan listrik rumah tangga dengan akses real-time via LCD dan web server

Deskripsi Proyek

Smart Energy Monitor adalah sistem cerdas yang dirancang untuk memantau penggunaan listrik dan mendeteksi lonjakan daya di rumah tangga. Sistem ini menampilkan data real-time melalui layar LCD dan dapat diakses melalui perangkat seluler via web server. Dengan alat ini, pengguna dapat memantau konsumsi energi, menerima notifikasi lonjakan listrik, dan mengoptimalkan penggunaan listrik untuk menghemat biaya.

Spesifikasi Teknis

Microcontroller

ESP32/Arduino - Pengolah Data

Sensor Arus

ACS712/Current Sensor - Deteksi Lonjakan

Display

LCD 16x2 - Tampilan Real-time

Konektivitas

Wi-Fi Module - Web Server Access

Fitur Utama

Monitoring Real-time

Pemantauan konsumsi listrik secara real-time dengan update data setiap detik

Deteksi Lonjakan

Sistem peringatan otomatis saat terjadi lonjakan listrik berbahaya

Akses Multi-Platform

Dapat diakses via LCD lokal dan perangkat mobile melalui web server

Data Historis

Penyimpanan data penggunaan untuk analisis pola konsumsi
